Detailed Description

DoubleSlider is a QWidget providing two sliders on one ridge to define a range. The sliders can be moved by mouse interaction or by setMaxValue() and setMinValue() method. The range of the values is from 0.0 to 1.0 and stored in float variables. To avoid an overlay of the two sliders the active one shunts the inactive slider.
