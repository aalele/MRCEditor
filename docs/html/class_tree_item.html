◆ takeChild()

TreeItem * TreeItem::takeChild ( int  row,
TreeItem *  child = nullptr,
bool *  takeSuccess = nullptr 
)
noexcept

This function replace the old child node with a new one.

Parameters
rowThe row number of the old child node in the TreeItem
childThe new child node that will replace the old one.
takeSuccesssuccess flag pointer. If it's not a null pointer, the reference is set as true when substitution is successful otherwise it is set as false.
Returns
Return tje old child node pointer
Note
If row is larger than the child number, nothing will be done. takeSuccess will be set as false if it's not a nullptr. The ownership of the old child pointer is returned to the caller and its parent item is set as nullptr.
